The use of the Twitter API with this account is as follows:

  1. In my service, we will present a twitter contribution data view like github contribution(https://help.github.com/en/articles/viewing-contributions-on-your-profile). It is a service that allows you to check your contribution in twitter app looks like github contribution looks and enjoy how much you can contribute in twitter app. We want to adopt api to realize these. Users using this service will enjoy watching their contributions by entering their own twitter ID. We do not maliciously use the obtained user data.

  2. There are no plans to retweet through Twitter APIs.

  3. There are no plans to favorite through Twitter APIs.

  4. As stated above, we swear that we will not perform any unauthorized operations through the twitter api.

We never publish data obtained through API directly to the outside. When publishing, we promise to process it into a guthub contribution format that users can enjoy like this > https://github.com/ptmt/twitter-contribution-chart >>>>>>> 90d8d19298236a2034bf6baa0ca6a51a51b5ca1b
