「ネットワークはなぜ繋がるのか」を読んだ

Aug 13, 2019

はじめに

業務で AWS EC2 を用いてWEBアプリケーションを作成した。 その際EC2の設定を触っていく中で分からない言葉が多かったので、書籍を購入したので読んで見る。

#

はじめに

冒頭にある「探検ツアーの流れ」というロードマップが分かりやすい。 パケットが小包という表現はわかりやすかった。 知っている言葉と知らない言葉でごちゃっている感。

第1章

URL末尾の「/」についてはなんとなく理解していたつもりだったものの、 活字で読んでみるとなるほどなぁ、という一面はあった。

JavaScriptでdecodeURIとかよく出てきたけど、URIとURLの違いについてはあまり意識していなかった。 なるほど感。

引き続き読む。

33ページにいきなり誤植が出てきて笑った。 「メゾッッド」

フォームのデータを渡すプログラムのファイル名という欄に、 .cgiというファイルがあった。これがhttpの送信を行っているファイルなのだろうか。そういえばfastCGIとか聞いたことある。

fastCGIはここがわかりやすかった。

nginxでphpを扱う際に、php-fpm(fastCGI process manager)を確認した。

あとからcgiについても勉強する。

個人的には業界柄この説明、特にCNAMEとかの開設が欲しかった。 ただ書籍的には省くのはわかる。

動画Libraryを作成する。 そしてMPEGとHLSで作成する。 ↓Libraryを作成するなど。

爆速で凄い人になること。

まずはrootを設定し直す。

サーバーの PHP バージョンは 5.3.29 ですが WordPress 5.2.2 は 5.6.20 以上のみでご利用になれます。
